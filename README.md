# Material Chips with options

This project shows an implementation/example of materialize.css chips with additional feature.

# Main features

  - Materializecss autocomplete chips have been limited to several options (you are free to type but not to select)
  - You can't repeat an added option
  - Everything is configured from the outer compoenent... cool!
  - Fully reusable

# Getting Started
1. Install angular-cli
2. Clone the git repository of the project.
3. Download and install latest version of [node](https://nodejs.org/es/download/)
4. Go to the main project folder

    ```
    cd /your_projects_path/your_project/
    ```

4. Install dependencies

    ```
    npm install
    ```
5. Start server and launch in the browser

    ```
    npm start
    ```

### Prerequisites

The project has the structure of an Angular-Cli project, so it is highly recommend to install and launch with it.
